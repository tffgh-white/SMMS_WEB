## 🌟 系统简介

iStock智能股票管理系统是一个专业的股票数据管理平台，为用户提供实时股票信息查询、收藏管理、个人账户管理等功能。系统界面简洁友好，操作简单直观。

### 主要功能
- 📈 **实时行情** - 查看沪深两市股票实时交易数据
- ⭐ **收藏管理** - 收藏关注股票，快速查看
- 👤 **账户管理** - 个人信息管理和设置
- 📊 **数据可视化** - 直观的图表展示
- 🔍 **股票搜索** - 快速搜索和筛选股票

## 🚀 快速入门

### 系统要求
- **浏览器要求**：Chrome 80+、Firefox 75+、Safari 13+、Edge 80+
- **网络要求**：稳定的互联网连接
- **屏幕分辨率**：推荐1920x1080或以上

### 首次使用

#### 1. 访问系统
在浏览器中输入系统地址：
```
https://istock.example.com
```

#### 2. 用户注册
- 点击页面右上角"注册"按钮
- 填写注册信息：
  - 用户名：3-20个字符，支持字母、数字、下划线
  - 邮箱：有效的邮箱地址
  - 密码：8-20个字符，包含字母和数字
- 点击"注册"完成账户创建

#### 3. 用户登录
- 点击页面右上角"登录"按钮
- 输入用户名和密码
- 点击"登录"进入系统

#### 4. 浏览股票
- 登录后进入首页
- 系统自动显示热门股票列表
- 可以浏览股票基本信息和实时价格

## 📖 功能详解

### 1. 首页概览

#### 主要区域
- **搜索栏**：快速搜索股票代码或名称
- **热门股票**：显示市场热门股票
- **实时指数**：上证指数、深证成指等
- **快速导航**：各功能模块快速入口

#### 操作说明
```
1. 搜索股票
   在搜索框输入股票代码（如：000001）或名称（如：平安银行）
   按回车键或点击搜索按钮

2. 查看股票详情
   点击任意股票卡片查看详细信息
   包含实时价格、涨跌幅、成交量等

3. 收藏股票
   点击股票卡片右上角的星标图标
   金色星标表示已收藏，灰色表示未收藏
```

### 2. 股票搜索功能

#### 搜索方法
1. **代码搜索**：直接输入股票代码
   - `000001` - 平安银行
   - `600000` - 浦发银行
   - `300001` - 特锐德

2. **名称搜索**：输入公司名称关键词
   - `平安` - 平安银行
   - `银行` - 所有银行类股票
   - `科技` - 科技类股票

3. **模糊搜索**：支持部分匹配
   - `平安银` - 平安银行
   - `银` - 所有包含"银"的股票

#### 搜索结果
```
搜索结果页面显示：
- 股票代码和名称
- 当前价格
- 涨跌幅（涨红跌绿）
- 成交量
- 所属交易所

点击任意结果查看详细信息
```

```
- 股票代码：如 000001.SZ
- 股票名称：如 平安银行

```


#### 图表功能
```
- K线图：显示价格走势

- 成交量图：成交量变化
- 技术指标图：各种技术指标
- 支持时间范围选择：1日、5日、1月、3月、6月、1年
```

#### 交互操作
```
1. 添加收藏
   点击"收藏"按钮添加到个人收藏列表

4. 导出数据

```

### 4. 收藏管理

#### 收藏列表
```
显示用户收藏的所有股票：
- 股票基本信息
- 实时价格和涨跌幅
- 收藏时间
- 快速操作按钮
```

#### 收藏操作
```
1. 添加收藏
   在股票详情页点击"收藏"按钮
   在搜索结果页点击星标图标
   在首页点击快速收藏按钮

2. 取消收藏
   在收藏列表点击"取消收藏"
   在股票详情页点击"已收藏"按钮

3. 批量操作
   勾选多个股票进行批量取消收藏
   支持全选和反选操作

4. 收藏分组
   创建收藏分组（如：科技股、银行股）
   将股票移动到不同分组
   重命名和删除分组
```

#### 收藏排序
```
支持多种排序方式：
- 按收藏时间（最新/最早）
- 按股票代码（升序/降序）
- 按当前价格（从高到低/从低到高）
- 按涨跌幅（涨幅最大/跌幅最大）
- 自定义排序（拖拽调整顺序）
```

## ❓ 常见问题

### 账户相关问题

#### Q1: 忘记密码怎么办？
**A**: 
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件中的重置链接
5. 设置新密码并确认

#### Q2: 如何修改用户名？
**A**:
1. 登录后进入"个人中心"
2. 点击"基本信息修改"
3. 输入新用户名
4. 输入密码确认身份
5. 点击"保存"完成修改

#### Q3: 可以更改邮箱地址吗？
**A**: 
可以，步骤如下：
1. 进入个人中心
2. 点击"更新邮箱"
3. 输入新邮箱地址
4. 输入密码验证
5. 查收新邮箱的验证邮件
6. 点击验证链接完成更改

### 功能使用问题

#### Q4: 股票数据多久更新一次？
**A**:
- 实时数据：交易时间内每5秒更新一次
- 基础信息：每日更新一次
- 历史数据：每日收盘后更新
- 指数数据：实时更新

#### Q5: 为什么有些股票搜索不到？
**A**: 
可能的原因：
- 股票已退市
- 输入的代码不正确
- 股票不在支持的交易所范围内
- 系统数据同步延迟

**解决方案**：
- 核对股票代码准确性
- 确认股票仍在交易
- 稍后重试搜索

#### Q6: 收藏数量有限制吗？
**A**:
- 普通用户：最多收藏200只股票
- VIP用户：最多收藏500只股票
- 如需更多收藏，请联系客服

### 技术问题

#### Q7: 页面加载缓慢怎么办？
**A**:
**网络问题**：
- 检查网络连接
- 清除浏览器缓存
- 尝试刷新页面

**浏览器问题**：
- 更新浏览器到最新版本
- 关闭不必要的浏览器插件
- 尝试使用其他浏览器

#### Q8: 图表显示不正常？
**A**:
**排查步骤**：
1. 检查浏览器是否支持Canvas
2. 清除浏览器缓存
3. 检查JavaScript是否启用
4. 尝试刷新页面

**Chrome浏览器检查**：
- 按F12打开开发者工具
- 查看Console错误信息
- 截图反馈给技术支持

#### Q9: 手机端如何使用？
**A**:
- 系统支持响应式设计
- 在手机浏览器中访问相同网址
- 界面会自动适配手机屏幕
- 支持触摸操作和手势

## 🔧 故障排除

### 常见错误及解决方法

#### 1. 登录问题
```
错误：用户名或密码错误
解决：检查大小写，重置密码

错误：账户被锁定
解决：等待30分钟后重试，或联系客服

错误：验证码错误
解决：刷新验证码重新输入
```

#### 2. 数据显示问题
```
错误：数据加载失败
解决：检查网络，刷新页面

错误：图表显示异常
解决：更新浏览器，清除缓存

错误：搜索无结果
解决：检查输入，确认股票状态
```

#### 3. 功能异常
```
错误：收藏失败
解决：检查收藏数量限制，稍后重试

错误：导出失败
解决：检查浏览器下载设置

错误：设置保存失败
解决：检查输入格式，重新提交
```
